### Dependencies

- React 16
- React Router
- Reactstrap
- Bootstrap

---
### How to Setup 

**Clone repository**
```
git clone https://github.com/dcmpa/mpa-client
```

**Install dependencies**
```
npm install
```

**Run the development server**
```
npm start
```
